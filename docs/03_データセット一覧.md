# 利用可能データセット一覧

出典: https://challenge2026.odpt.org/ja/opendata.html（2026-07-02取得）
提供データは準備が整い次第随時追加される。

## 必須データ（どちらかの利用が応募条件）

### 1. 公共交通オープンデータセンター（ODPT）
- カタログ: https://ckan.odpt.org/dataset ／ 開発者サイト: https://developer.odpt.org/
- **チャレンジ限定提供データあり**（エントリー後にアクセス可）

| モード | 内容 |
|---|---|
| 鉄道 | JR東日本のGTFS時刻表、**GTFS-RT列車ロケーション・運行情報**ほか多数（ODPT独自JSON APIも存在） |
| バス | 都営バス等のGTFS時刻表、**GTFS-RTバスロケーション** |
| デマンド交通 | **GTFS-Flex形式**（2026の目玉。順風路/SWAT/MONET＋9自治体が協力） |
| フェリー | GTFS形式の航路情報 |
| 航空 | JAL/ANAのフライト時刻表、リアルタイム発着情報 |
| シェアサイクル | ドコモ・バイクシェア、OpenStreet（ハローサイクリング）の**GBFS形式**ポート情報 |

### 2. GTFSデータリポジトリ
- https://gtfs-data.jp/ ／ 検索: https://gtfs-data.jp/search
- 全国のコミュニティバス・民間バス・鉄道・フェリーのGTFS。**時系列アーカイブをAPI/ダウンロードで取得可**。オープンライセンス

## データ形式メモ

- **GTFS**: 静的時刻表（stops/routes/trips/stop_times）
- **GTFS-RT**: リアルタイム（車両位置 VehiclePosition / 遅延 TripUpdate / 運行情報 Alert）
- **GTFS-Flex**: エリア型・予約型のデマンド交通を表現する拡張（location groups / booking_rules 等）。対応する経路検索実装は世界的にもまだ少ない
- **GBFS**: シェアサイクルのポート・台数リアルタイム
- ODPT独自API: odpt:Station, odpt:Train, odpt:TrainInformation 等のJSON-LD（東京圏の鉄道に厚い）
