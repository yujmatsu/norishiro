# 公共交通オープンデータチャレンジ2026 開催概要・募集要項

- 公式サイト: https://challenge2026.odpt.org/
- 開催概要: https://challenge2026.odpt.org/ja/outline.html
- エントリー: https://challenge2026.odpt.org/ja/entry.html
- 情報取得日: 2026-07-02

## コンテストの趣旨

公共交通オープンデータを活用したオープンイノベーションのためのアプリケーションコンテスト。鉄道・バス・デマンド交通・航空・フェリー・シェアサイクルのデータを対象とする。

**2026年の重点（トップページの開催趣旨より）**
- Project LINKS / COMmmmONS 連携で公開された**デマンド交通データ**（GTFS-Flex）
- ほこナビプロジェクトによる**バリアフリー関連データ**の充実化
- 今年の新規取組: **地方都市圏を含む交通データの拡充**、**駅構内図データの充実化**
- 想定する社会課題: **少子高齢化、交通空白、オーバーツーリズム、バリアフリー対応**

## 募集内容

公共交通オープンデータを活かした「これまでにない新しいアプリケーションやサービス」。地方での課題解決や新しいデータ利活用につながるもの。

- **必須**: 公共交通オープンデータセンターおよびGTFSデータリポジトリで公開されている公共交通オープンデータ（鉄道・バス・デマンド交通・フェリー・航空・シェアサイクル等）の活用
- **強く推奨**: ほこナビ / Project LINKS / Project PLATEAU のオープンデータ活用
- **推奨**: オープンデータ・パートナー提供データの活用
- 応募形態: アプリケーション、Webサービス、IoTガジェット等

## 表彰

- 賞金総額: **最大300万円**
- 協賛組織からの特別賞も設置予定（過去実績: JR東日本賞、Project LINKS賞、INIAD賞 等）
- 賞の種類・受賞数・賞金額は審査会で決定

## 応募資格

- 個人・グループ・法人を問わず誰でも応募可能（国籍・年齢・居住地の制限なし）
- 1応募者で複数作品の応募可、複数グループへの所属可

## 応募の流れ

1. 公共交通オープンデータセンター開発者サイト（https://developer.odpt.org/）にユーザ登録（確認に最大2営業日）
2. チャレンジ2026にエントリー（→ チャレンジ限定データへのアクセスが可能になる）
3. 作品を開発
4. 応募期間中に開発者サイトから応募（応募フォームは2026年10月1日公開予定）

## 応募条件・規約の要点（後の評価・運営で特に重要なもの）

- 公共交通オープンデータセンター / GTFSデータリポジトリのデータを**必ず利用**すること
- 成果物は**コンテスト期間中、誰でも無償で利用できる状態で公開**すること
- iOS/Androidアプリはストア掲載の場合、**締切までにリリース済み**であること（審査登録日＝応募日。締切後の審査に影響する修正は無効）
- API利用規約・ガイドライン、各データのライセンスを遵守
- 応募作品は自作に限る。著作権処理は開発者が保証。知的財産権は開発者に帰属
- 開発・公開費用は開発者負担
- 一次審査対象者はヒアリングに出席、受賞者は表彰式に参加すること
- 作品内に**問い合わせ先を明記**（データ提供者へ問い合わせが行かないように）
- チャレンジ用データの目的外利用・第三者提供の禁止
- データ提供はチャレンジ終了までの予定（提供中止・内容変更の可能性あり）
